
Interest Calculator (Java) :

This project simulates a basic "Bank Deposit Service" using a Java console application.  
It allows customers to enter deposit details such as principal amount, tenure, gender, and age.  
Based on the provided information, the system calculates the total deposited amount and the interest earned.
The application demonstrates how banks compute interest using simple financial formulas and eligibility conditions

Technologies Used : 
• Java (JDK 8+)
• Eclipse IDE
• Scanner class for user input
